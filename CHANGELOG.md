# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-26

### Added
- Initial release
- Section-level semantic diff for Markdown documents
- Rich terminal output with colored panels
- Inline diff display for modified sections
- Section movement detection
- Git commit diff integration (`mddiff commit`)
- Summary statistics output
- Section filtering (`--section`)
- Python 3.9+ support
